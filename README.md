




## 📒 InventoryShare, [인벤토리 공유 플러그인]
> 마인크래프트 서버 플러그인

---

#### 📖 어떤 플러그인인가요?
지정된 플레이어간의 인벤토리를 공유하고 사용 가능 도구를 제한할 수 있는 플러그인입니다.

#### 📄 명령어
```
* /invenshare <닉네임> - 인벤토리 공유 목록에 해당 플레이어를 추가/해제합니다.
* /제한 <칼/도끼/삽/괭이/곡괭이> - 사용 가능 도구를 제한합니다.
```

#### 📄 Config

```
별도의 Config를 제공하지 않습니다.
```
